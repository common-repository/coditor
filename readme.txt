Coditor - Standalone code editor.

== Description ==
Most of code editors for Wordpress is just extending default editor of Wordpress. Coditor is different, instead of extending default editor, Coditor made its own editor, it can edit your plugins or themes. Coditor comes with nice Monokai theme.

== Installation ==
1.Upload your plugin to the '/wp-content/plugins/' directory

2.Activate the plugin through the 'Plugins' menu in WordPress.

You will find new menu named 'Coditor' in your WordPress admin panel.
